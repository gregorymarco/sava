# Sava

A turn-based chess-like web game built with Flask and HTMX.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the server:
```bash
python app.py
```

3. Open your browser and navigate to:
```
http://localhost:5000
```
